무작위 보행에서의 거리 제곱의 기댓값

목차

  • 기댓값 계산

A First Course in Probability, 9th edition by Sheldon Ross
Chapter 7, Example 2l, p.288

Example 2l 평면 위의 한 점에 처음 위치한 입자를 생각해 보자.
이 입자는 고정된 길이의 걸음을 반복적으로 밟는데, 각 걸음의 방향은 완전히 무작위이다.
구체적으로 말하면, 각 걸음 이후의 위치는 이전 위치로부터 거리 1만큼 떨어져 있으며,
그 방향은 이전 방향으로부터의 각이 \((0, 2\pi)\) 구간에서 균등하게 선택되는 것으로 가정한다.
이때, \(n\)번의 걸음 이후 원점으로부터의 거리의 제곱의 기대값을 구하라.

구해야 하는 것: 거리의 제곱 \(D^2\) 의 기댓값 \(\mathbb{E}[D^2]\)

난 이 문제의 답이 0 같은 게 아닐까 싶었다. 그런데 전혀 아니었다.
만약 1차원 수직선 상에서 \(\frac{1}{2}\)의 확률론 왼쪽 혹은 오른쪽만 갈 수 있다면 0일 것 같은 느낌은 든다.
(근데 이건 실제로 계산은 안 해 봤다.)

\(i\)-번째 걸음에서의 위치 변화를 \((X_i, Y_i)\) 라고 하면:
\(i = 1, \dots, n\)

\[ X_i = \cos \theta_i, \quad Y_i = \sin \theta_i \quad \text{where } \theta_i \sim \text{Unif}(0, 2\pi) \]

그러면 \(n\) 걸음 후의 위치의 좌표는 다음과 같다:

\[ \left( \sum_{i=1}^n X_i, \sum_{i=1}^n Y_i \right) \]

따라서 거리의 제곱은:

\[ D^2 = \left( \sum_{i=1}^n X_i \right)^2 + \left( \sum_{i=1}^n Y_i \right)^2 \]

이를 전개하면:

\[ \begin{aligned} D^2 &= \left( \sum_{i=1}^n X_i \right)^2 + \left( \sum_{i=1}^n Y_i \right)^2 \qquad \text{(피타고라스 정리)} \\ &= \sum_{i=1}^n (X_i^2 + Y_i^2) + \sum_{i \ne j} (X_i X_j + Y_i Y_j) \qquad \text{(제곱 전개)} \\ &= n + \sum_{i \ne j} \left( \cos \theta_i \cos \theta_j + \sin \theta_i \sin \theta_j \right) \\ &\qquad \text{(각 $\theta_i$에 대해 $\cos^2\theta_i + \sin^2\theta_i = 1$이므로} \\ &\qquad \text{$\sum_{i=1}^n (X_i^2 + Y_i^2) = n$)} \end{aligned} \]

기댓값 계산

이제 기대값을 취하자:

\[ \begin{aligned} \mathbb{E}[D^2] &= \mathbb{E} \left[ n + \sum_{i \ne j} \left( \cos\theta_i \cos\theta_j + \sin\theta_i \sin\theta_j \right) \right] \qquad \text{(아까 구한 $D^2$ 식에 기댓값 적용)} \\ &= \mathbb{E}[n] + \sum_{i \ne j} \mathbb{E}[\cos\theta_i \cos\theta_j] + \mathbb{E}[\sin\theta_i \sin\theta_j] \qquad \text{(선형성: $\mathbb{E}[X+Y] = \mathbb{E}[X] + \mathbb{E}[Y]$)} \\ &= n + \sum_{i \ne j} \mathbb{E}[\cos\theta_i] \mathbb{E}[\cos\theta_j] + \mathbb{E}[\sin\theta_i] \mathbb{E}[\sin\theta_j] \qquad \text{(독립성 활용: $\mathbb{E}[XY] = \mathbb{E}[X] \mathbb{E}[Y]$)} \\ &= n + \sum_{i \ne j} (0 + 0) = n \qquad \text{($\mathbb{E}[\cos\theta_i] = \mathbb{E}[\sin\theta_i] = 0$이므로)} \end{aligned} \]

마지막으로, 아래의 항에서 등장했던 \(\mathbb{E}[\cos \theta_i]\)와 \(\mathbb{E}[\sin \theta_i]\)가 왜 0이 되는지를 수식으로 확인해 보자:

\[ \begin{aligned} \mathbb{E}[\cos \theta_i] &= \frac{1}{2\pi} \int_0^{2\pi} \cos u \, du = \frac{1}{2\pi} (\sin 2\pi - \sin 0) = 0 \\ \mathbb{E}[\sin \theta_i] &= \frac{1}{2\pi} \int_0^{2\pi} \sin u \, du = \frac{1}{2\pi} (-\cos 2\pi + \cos 0) = 0 \end{aligned} \]

이는 \(\theta_i\)가 \([0, 2\pi)\) 구간에서 균등하게 분포한다고 가정할 때, \(\cos \theta_i\)와 \(\sin \theta_i\)의 평균이 각각 0이 됨을 보여준다.

직관적으로 생각해 보아도, \(y = \cos x\)와 \(y = \sin x\)는 \([0, 2\pi]\) 구간에서 \(y = 0\)을 기준으로 대칭적으로 위아래로 진동하기 때문에, 그래프의 평균값이 0이라는 사실은 쉽게 예상할 수 있다.

따라서 \[ \boxed{ \mathbb{E}[D^2] = n } \]

예상했던 것과 달리, 무작위 보행에서의 거리 제곱의 기댓값은 \(n\)이다.
왜냐하면 입자의 최종 위치는 완전히 랜덤하므로 방향 자체는 상쇄될 수 있지만,
거리는 걸음수가 늚에 따라 증가하기 때문이다.
즉, 무작위 보행에서는 방향은 상쇄되어도,
거리 제곱의 기댓값은 선형적으로 증가한다.
